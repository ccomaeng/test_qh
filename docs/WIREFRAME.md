## 11. 접근성 고려사항

- **키보드 네비게이션**: Tab 순서 최적화
- **스크린 리더**: aria-label, role 속성 적용
- **색상 대비**: WCAG 2.1 AA 준수
- **폰트 크기**: 최소 16px, 확대 가능
- **에러 메시지**: 명확하고 구체적인 안내
- **진행 상황**: 시각적/청각적 피드백 제공

이 wireframe은 실제 구현된 시스템의 사용자 인터페이스를 체계적으로 설계한 것으로, 향후 UI/UX 개선이나 새로운 기능 추가 시 참고할 수 있는 기본 설계도 역할을 합니다.
